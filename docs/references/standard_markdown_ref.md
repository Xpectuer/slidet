
# Markdown 标准语法参考指南

本指南涵盖了标准 Markdown（CommonMark）以及最常用的扩展语法（如 GitHub Flavored Markdown）。

## 1. 标题 (Headings)
使用 `#` 号来表示标题，`#` 的数量代表标题的级别（1-6级）。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 2. 文本强调 (Emphasis)
用于设置斜体、加粗和删除线。

| 效果 | 语法 | 示例 |
| :--- | :--- | :--- |
| *斜体* | `*文本*` 或 `_文本_` | `*斜体文本*` |
| **加粗** | `**文本**` 或 `__文本__` | `**加粗文本**` |
| ***粗斜体*** | `***文本***` | `***又粗又斜***` |
| ~~删除线~~ | `~~文本~~` | `~~已作废的内容~~` |

## 3. 列表 (Lists)

### 无序列表
使用 `*`、`+` 或 `-` 作为列表标记。
```markdown
- 项目一
- 项目二
  - 子项目 A
  - 子项目 B
```

### 有序列表
使用数字加英文句点。
```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 任务列表 (Task Lists)
```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

## 4. 链接与图片 (Links & Images)

### 链接
语法：`[显示文本](URL "可选标题")`
```markdown
[百度一下](https://www.baidu.com)
```

### 图片
语法：`![替代文字](图片地址 "可选标题")`
```markdown
![Markdown Logo](https://markdown-here.com/img/icon256.png)
```

## 5. 引用 (Blockquotes)
使用 `>` 号。
```markdown
> 这是一个引用块。
> 它可以跨越多行。
>> 甚至可以嵌套引用。
```

## 6. 代码 (Code)

### 行内代码
使用单个反引号 `` ` ``。
```markdown
在句子中使用 `printf()` 函数。
```

### 代码块
使用三个反引号 ` ``` `，并可指定语言进行语法高亮。
```markdown
```python
def hello_world():
    print("Hello, Markdown!")
```
```

## 7. 分割线 (Horizontal Rules)
使用三个或更多的 `*`、`-` 或 `_`。
```markdown
---
***
___
```

## 8. 表格 (Tables)
使用 `|` 分隔列，使用 `-` 分隔表头和内容，使用 `:` 设置对齐方式。
```markdown
| 姓名 | 年龄 | 职业 |
| :--- | :--: | ---: |
| 张三 | 25 | 工程师 (左对齐) |
| 李四 | 30 | 设计师 (居中) |
| 王五 | 28 | 产品经理 (右对齐) |
```

## 9. 转义字符 (Escaping)
如果你想在文档中显示 Markdown 语法使用的特殊字符，请在前面加上反斜杠 `\`。
```markdown
\* 这不是列表，而是一个普通的星号。
```

## 10. 数学公式 (LaTeX)
许多现代编辑器支持 LaTeX 语法。
- 行内公式：`$E=mc^2$`
- 块级公式：
$$
\begin{aligned}
a^2 + b^2 = c^2
\end{aligned}
$$
